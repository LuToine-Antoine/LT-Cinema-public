## Sommaire

- I - Prérequis
- II - Initialisation

## I - Prérequis

Afin de pouvoir ouvir le site et d'avoir accès à la totalité des fonctionnalités, veuillez installer l'extention live server sur votre environnement de programmation.
Vous devrez ensuite vous créer un compte sur l'[api TMDB](https://www.themoviedb.org/?language=fr) et vous créer une clef API.

### Pour créer une clef API :

Rendez-vous dans VOtre profil -> paramètres -> API.

### Mettre la clef dans le programme.

Pour mettre que le programme fonctionne, rendez vous dans sso.js et mettez votre OKEN API et non la clef API dans la constante :
```js const ssoTmdbReadApiKey```.

## II - Initilisation

Pour lancer le site, veuillez ouvrir la page index.html en premier.
Ensuite, une fois que vous aurez installer l'extention live server, vous devrez faire clique droit dans la page index.html et selectionner "open with live server".
Vous n'avez plus qu'à vous balader sur le site :) .
